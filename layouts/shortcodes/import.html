{{- $path := .Get "path" -}} {{- $content := readFile $path | htmlUnescape | markdownify -}} {{- $content := split $content "\n" -}} {{- $start := .Get "from" | default "1" | int | add -1 }} {{- $end := .Get "to" | default (len $content) | int }} {{- $end := sub $end $start }} {{- $content := after $start (first (add $start $end) $content) -}} {{- $content := delimit $content "\n" -}} {{- $content -}}
